## Decision

```text
upper3072-v0:
  HOLD as small-v0 default

reason:
  phase peak RSS improvement is real and reproducible
  hot gates do not meet the >=0.98 median ratio requirement

p2-v0:
  remains the frozen small-v0 / rc1 default
```

`upper3072-v0` remains useful evidence for SizePolicy-v1.  It should not be
merged as default before a design decision changes the hot-gate threshold or a
new implementation recovers the local/interleaved loss.
